The Answers to Exercise 5

h2

This selector makes the contain of h2 tag has blue background color and opacity of 0.5. It means the text inside h2 is faded.


div

Sets to top edge of the positioned tag element 100px down from the top edge of its nearest positioned ancestor and 200px left between the element left edge and the left edge of its containing block.


.pos

Sets to top edge of the positioned tage element 150px down from the top edge of its nearest position ancestor.


table,th,td

creates a red border around a table, th, and td tags.


td.bg

Change the background color of each cell with the class of 'bg' to the color green.


td.bd

Change the border of each cell with a class of 'bd' to dashed.


th.padding

generate space 20px on the top and bottom of any cell with the class of padding.


.user1

aligns text right and changes the color to violet any one with a class of user1.


#user1

aligns text to the center and changes the font style to italic any tag with the id of user2.
